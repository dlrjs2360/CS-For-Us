# 💡 스터디 방식
- 분야를 정한 후 서로 다른 목차에 대해 각자 공부하고 마크다운으로 정리 후 깃허브에 올림.
- 금요일에 올린 마크다운 파일을 보면서 서로에게 공부한 내용을 설명함.
- 설명이 모두 끝나면 30분 동안 서로의 내용을 공부하고 각자 질문을 던져주기.
- 관련 면접 질문도 정리해오기.
